# 🏦 Smart Lender - Loan Eligibility Prediction System

## 📌 Project Description

Smart Lender is a Machine Learning-based web application that predicts whether a loan applicant is eligible for a loan. The application uses financial information such as applicant income, credit score, loan amount, and years of employment to generate loan eligibility predictions.

The project is developed using Python, Flask, XGBoost, HTML, CSS, and SQLite.

